## Netlify Deployment

1. Push your code to GitHub.
2. Connect your repository to Netlify.
3. Set the build command to `npm run build` and the publish directory to `.next` or `out` (for Next.js static export).
4. Add any required environment variables in the Netlify dashboard.

For more details, see the [Netlify Docs](https://docs.netlify.com/).

This is a NextJS starter in Firebase Studio.

To get started, take a look at src/app/page.tsx.
